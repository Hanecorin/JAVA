1. java의 데이터 타입
	기본 타입 : new로 데이터 미생성
		- 정수(0) : byte, short, int, long
		- 실수(0.0) : float, double 
		- 문자('\000') :char
		- 논리(false) : boolean
		
	참조 타입
		- null
		- class 타입, 객체 타입
		- new로 데이터 생성 (객쳬 생성)
		- 기본을 제외한 모든 타입
		- class와  interface가 있을 경우 type으로 사용 가능
		- API : String, Integer, Math ...
		- 배열
		- 참조 타입이긴 하나 두가지 type을 표기법 만으로 객체화 해서 사용 가능
			- new 키워드 없이 자동으로 객체가 되는 두가지 타입
			String -  " "
			배열 - {}
		
2. java 변수 선언 위치에 따른 구분
	멤버 변수
		- class {} 내부에 선언되는 변수 
		- heap에 저장, 기본값으로 자동 초기화 
		- instance 변수 : 객체 생성 후에만 사용 가능한 변수
		- static 변수 : 객체 생성 여부와 무관하게 사용 가능한 변수
			
	로컬 변수
		- 생성자, 메소드 () or {} 선언되는 변수
		- 생성되는 시점은 생성자, 메소드 호출시에 생성 및 종료 시 자동 메모리에서 소멸
		- 자동 초기화 되지 않음
