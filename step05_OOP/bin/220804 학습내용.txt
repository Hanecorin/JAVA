			java.lang package에 기본 데이터를 support하는 8가지 클래스 제공
				- wrapper class
					int - Integer
						문자열을 int로 변환하는 메소드
							int 변수 = Integer.parseInt("3");
							
						byte 변수 = Byte.parseByte("3");
						double 변수 = Double.parseDouble("3.0");
							
					boolean - Boolean
	
				- autoboxing & unboxing
					- 기본 타입과 wrapper class 사이에서만 적용되는 메카니즘
